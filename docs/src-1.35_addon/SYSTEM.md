# SYSTEM — システム全体像／用語／範囲

## 1. 目的
テキスト主導でストーリーボード（コンテ）を**高速・再現性・安全性**をもって作成する。

## 2. 構成
- **アプリ層（PWA）**: React/Vite。UIと軽量ローカル処理（mock生成・整形）。
- **core層**: 変換（MD/CSV/JSONL正規化）、精緻マージ、Golden比較、ID/seed生成。
- **platform I/F**: `fs / settings / dialog / net` の抽象。PWA→拡張/Qtへ差し替え可能。

## 3. 用語
- **カード**: 1カットを表す最小単位。title/desc/manual/ai/status 等を持つ。
- **生成ライン**: 右ペインの内容。v1.0は**モック**（テンプレ展開/要約/差分）で動作。
- **統合**: 下ペインで採用/棄却/編集を行い、主データへ取り込む行為。

## 4. 対象範囲（v1.0）
- UI 3ペイン、レイアウト4種、AutoSave、精緻マージ、Export/Import、Package Export、プラグインv1。

## 5. 非対象（後段）
- 実LLM呼び出し、IDE統合（拡張化）、Snippets双方向同期の完成形、クラウド同期、画像サムネ生成。
