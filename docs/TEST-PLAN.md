# テスト計画（v1.3）

## 1) 受入テスト（必須）
1. Seed再現：同一 seed/設定でテキスト一致率 ≥ 99%（空白差は無視）。
2. Seed非対応：`seed_applied:false` が evidence に出る。
3. AutoSave：停止後 2.5s 以内に保存／異常終了後の復旧成功。
4. 履歴ローテーション：21回保存後に最古が削除。
5. 3-wayマージ：
   - ラベルあり：自動マージ成功。
   - 類似度 < 0.75：衝突が残る。
   - 衝突解消後の再実行で結果不変。

## 2) E2E（推奨）
- 10カット作成 → 生成（seed指定）→ マージ → Snapshot保存 → 復元 → Export/Import ラウンドトリップ。
